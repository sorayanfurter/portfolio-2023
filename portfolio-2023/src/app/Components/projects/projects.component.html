TEST PORTAL

Built using Angular/Java and designed for creating and taking multiple-choice exams. Admins will be have able to create exams, add questions with their respective scores, and set time limits for users to answer. Additionally, admins can organize the exams into different categories and create new ones. On the other hand, users will be able to take the exams available to them.
